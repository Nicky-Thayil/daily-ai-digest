# Flash

## Overview
Flash is a backend-focused service designed to provide daily information and summaries about my favorite topics.

## Features
- Topic configuration via JSON (topic generation/addition will be added later)
- Provides daily summaries and information about favorite topics

## Tools & Technologies
- Python
- FastAPI
- OpenAPI
- SQLite (may migrate to Supabase/Postgres)
